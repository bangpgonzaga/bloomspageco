# BloomsPage Co. Landing Page

A modern, responsive landing page to showcase your products.

## Features

- ✨ Modern and responsive design
- 🎨 Beautiful gradient backgrounds
- 📱 Mobile-friendly layout
- ⚡ Smooth animations and transitions
- 🛒 Add to cart functionality
- 📧 Contact form
- 📊 Product showcase grid

## Getting Started

1. Clone the repository
2. Open `index.html` in your browser
3. Customize the products and content as needed

## File Structure

- `index.html` - Main HTML structure
- `styles.css` - All styling and responsive design
- `script.js` - Interactive features
- `README.md` - Documentation

## Customization

### Add Your Products

Edit the product cards in `index.html` by updating:
- Product name (in `<h3>` tags)
- Product description
- Product price

### Change Colors

Update the gradient colors in `styles.css`. Current primary gradient:
```css
linear-gradient(135deg, #667eea 0%, #764ba2 100%)
```

### Update Company Info

Replace "BloomsPage Co." and update contact information as needed.
